# coeur-soleil-site

Public marketing website for **Distribution Coeur-Soleil** (FR / EN / ES).

Single-file static site (`index.html`), deployed on Vercel. The build gate runs `node check.js index.html` to fail on any inline-script syntax error, mirroring the CRM/app repos.

- Home / Pour les hôtels / Pour les partenaires (hash-routed)
- Trilingual language switcher, scroll-reveal motion
- Logo drawn in-canvas (heart-sun mark)
